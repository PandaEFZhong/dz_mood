# 酷安上架材料包

## 1. 开发者注册

访问：https://developer.coolapk.com/

- 用酷安账号登录（没有就注册一个）
- 进入「开发者中心」→「发布应用」
- 个人开发者直接上传，**不需要企业认证**

### 应用名称

### 一句话简介

### 应用介绍
```
乐心是一款简洁优雅的心情记录与健康关怀应用。

【核心功能】
• 心情日记：用文字记录当下的情绪与感受
• AI 智能分析：基于 Kimi AI 分析心情内容，给出情绪评分与身体负荷指数
• 健康建议：AI 根据记录内容提供个性化的身体调节建议
• 数据可视化：7天心情趋势图、情绪分布统计，直观了解心理状态变化
• 每日提醒：晚间定时提醒，养成记录习惯

【适用场景】
• 工作压力大，想理清情绪头绪
• 关注身心健康，希望建立自我觉察习惯
• 需要简单的工具追踪情绪与身体状态变化

【隐私保护】
所有数据仅存储在本地设备，不上传服务器。详细隐私政策见：https://pandaefzhong.github.io/dz_mood/privacy-policy.html

【声明】
本产品仅为健康监测和自我管理工具，不能替代专业医疗诊断。如有身体不适，请及时就医。
```

## 3. 应用截图要求

需要 **3-5张** 应用真实界面截图，要求：

- **尺寸**：1080×1920 或 1080×2340（竖屏，手机原图即可）
- **内容**：展示应用的实际使用界面，不要过度修饰
- **建议画面**：
  1. **记录页**：填写心情内容后的界面（展示 AI 分析结果）
  2. **历史页**：有 3-5 条记录的列表（展示 MoodCard 效果）
  3. **统计页**：图表数据展示（折线图 + 饼图）
  4. **展开效果**：点击展开一条长记录，展示「展开全文」效果
  5. **整体布局**：展示底部 Tab 切换

### 截图操作建议
在真机上操作：
1. 先记录 5-7 条不同的心情（内容长短搭配，有带身体数据的有不带的）
2. 等一天再截图，这样历史页内容更丰富（或者修改手机时间也行）
3. 截图时关闭「开发者选项」中的布局边界等调试标记（设置→开发者选项→关闭「显示布局边界」）
4. 截图前清理通知栏，保持状态栏干净

## 4. APK 文件

https://github.com/PandaEFZhong/dz_mood/actions

或本地构建：
```bash
cd android && ./gradlew assembleRelease
```

APK 路径：`android/app/build/outputs/apk/release/app-release.apk`

## 6. 酷安上传步骤

1. 登录 https://developer.coolapk.com/
2. 点击「发布新应用」
3. 上传 APK
4. 填写应用信息（复制上面的内容）
5. 上传截图
6. 提交审核

酷安审核通常 **1-3 个工作日**，比小米宽松很多。

---

## 7. 注意事项

- 酷安社区属性强，用户喜欢「真实、简洁、有用」的应用，文案不要过度营销
- 可以在应用介绍最后加一句：「独立开发，欢迎反馈建议」——酷安用户很吃这套
- 上架后可以在酷安发动态介绍你的应用，增加曝光
