# 乐心 (HeartMate) 首发推广方案

## 产品定位

**一句话介绍**：AI 心情记录 + 身体负荷监测，帮你听懂身体的信号。

**目标用户**：
- 25-35 岁职场人，压力大、容易焦虑
- 关注身心健康但不想用复杂工具的人
- 有记录习惯（日记、手账）的人

**核心卖点**：
1. 🧠 AI 深度分析 —— 不只是打分，告诉你「为什么」和「怎么办」
2. 💓 身体负荷指数 —— 把情绪和身体状态关联，发现隐藏模式
3. 🔒 隐私优先 —— 本地存储，云端同步可选，数据你自己掌控
4. 📊 可视化趋势 —— 周/月统计，看见自己的变化
