## 一、身体负荷指数（Body Load Index）

### 1.1 是什么？

AI 综合你输入的 **静息心率 + 睡眠质量 + 疲劳感 + 情绪状态**，算出的一个身体负担分数（1-10）。

**不是医学诊断**，是一个帮你快速感知"身体现在有多累"的参考指标。

### 1.2 评分标准（AI 判断逻辑）

| 分数 | 状态 | 颜色 | AI 判断依据 |
|:---:|---|:---:|---|
| 1-3 | 😌 轻松 | 🟢 | 心率正常 + 睡眠好 + 疲劳低 |
| 4-6 | 😐 适中 | 🟡 | 个别指标略高，但整体可控 |
| 7-8 | 😰 较重 | 🟠 | 多项指标同时亮黄灯 |
| 9-10 | 🚨 过载 | 🔴 | 心率过快 / 极度疲劳 + 差睡眠 |

### 1.3 展示位置

#### 位置 A：提交成功提示卡片（最优先）

用户刚提交完记录，最想知道"我现在的身体状态怎么样"。

**UI 设计：**
```
┌────────────────────────────────────────┐
│  🫀 身体负荷  [████████░░]  8/10        │
│           😰 较重                       │
│                                        │
│  你已连续 2 天心率偏高 + 睡眠不足，     │
│  身体正在发出求援信号。                │
└────────────────────────────────────────┘
```

- 用横向进度条展示，直观
- 分数+状态文字放在右侧
- 背景色根据分数变化（绿/黄/橙/红）
- 如果触发了预警，身体负荷条放在预警卡片下方

#### 位置 B：历史记录卡片（每条记录上）

回顾历史时，能快速扫一眼那天的身体状态。

**UI 设计：**
- 紧凑形式，放在身体指标标签（🫀 80 bpm / 😴 睡眠 5/10）的下方
- 只显示进度条 + 分数，不展开详细说明

```
🫀 80 bpm 正常  😴 睡眠 5/10  😫 疲劳 5/10
[████░░░░░░] 身体负荷 4/10
```

#### 位置 C：统计页面（后续做图表时接入）

- 身体负荷 7 天趋势折线图
- 和情绪分数放一起双轴对比

### 1.4 交互细节

- **hover 提示**：鼠标/手指悬停在进度条上，显示"身体负荷是什么？"的简单说明
- **无身体指标时隐藏**：如果用户没输入心率/睡眠/疲劳，不显示身体负荷（AI 无法计算）

## 二、恢复建议（Body Advice）

### 2.1 是什么？

AI 看了你的身体指标后，专门从**身体层面**给你的建议。

和现有的「💡 贴心建议」的区别：

| 建议类型 | 关注点 | 示例 |
|---|---|---|
| 💡 贴心建议（已有） | 心理/情绪层面 | "可以尝试冥想、瑜伽来改善情绪" |
| 🫀 恢复建议（新增） | 身体/生理层面 | "你的心率已连续 2 天偏高，建议今晚 10 点前睡觉，明天减少咖啡因摄入" |

### 2.2 展示位置

#### 位置 A：提交成功提示卡片

放在「💬 洞察」和「💡 贴心建议」之间，或者最下方。

**UI 设计：**
```
┌────────────────────────────────────────┐
│  💬 洞察：你在离职休息期间...            │
├────────────────────────────────────────┤
│  🫀 身体建议：你的心率已连续 2 天偏高，   │
│     建议今晚 10 点前睡觉，明天减少咖啡   │
│     因摄入。                            │
├────────────────────────────────────────┤
│  💡 贴心建议：在寻找新工作的同时...       │
└────────────────────────────────────────┘
```

- 用浅绿色/浅蓝色背景，和橙色建议卡片区分
- 标题固定为「🫀 身体建议」
- 文字精简，2 句话以内

#### 位置 B：历史记录卡片

- 放在「💬 洞察」和「💡 贴心建议」之间
- 如果用户没输入身体指标，不显示这个模块

### 2.3 内容规范（AI Prompt 约束）

要求 AI 按这个格式输出恢复建议：

- **身体负荷低（1-3）**：鼓励保持，给维持建议
- **身体负荷中（4-6）**：提醒注意休息，给具体作息建议
- **身体负荷高（7-8）**：强烈建议减少工作强度，给具体行动清单
- **身体负荷过载（9-10）**：建议就医检查，暂停高强度工作
